# Retail Performance Intelligence Dashboard

A consulting-style analytics portfolio project built in Python and Streamlit using supermarket transaction data.

## Project purpose
This project demonstrates end-to-end data analysis capability: data auditing, feature engineering, KPI design, statistical testing, interactive storytelling, and executive-ready reporting.

## What the dashboard shows
- Executive KPI overview
- Revenue by product line
- Monthly sales trends
- Customer-type spending comparisons
- Payment-method performance
- Product-line ratings
- Seasonality heatmap
- Branch positioning
- Data quality audit and CSV export for filtered views

## How to run locally
```bash
pip install -r requirements.txt
streamlit run dashboard/app.py
```

## Deployment
This project is ready for Streamlit Community Cloud, Render, or a VPS deployment.
Use:
- `dashboard/app.py` as the main app file
- `requirements.txt` for environment setup
